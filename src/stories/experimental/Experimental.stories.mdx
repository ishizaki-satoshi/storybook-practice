# Experimentalについて
新しく技術を取り入れる際の、検討・ブラッシュアップに利用

### 含めるもの
- 試行錯誤中の技術を利用したスタイル・アニメーション

### 目的
- 他の人のレビュー
- 技術的アドバイス
- Showcaseに移動するためにデザイナーにスタイルをつけてもらう
- 最終的にはShowcaseに移動

### ターゲット
- 使う人
  - エンジニア
  - （デザイナー）
- 使う場面
  - 新しい技術の共有・相談
  - （具体的な利用シーン・デザインを考える）

### 補足
- 他のパーツと同じにして「READY」「WIP」などのラベルを出し分けられるようにする対応でもいいかもしれない
- 同じリポジトリである必要性はない
- 必ずしもStorybookである必要性はない
